# manhole

A shitty, temporary Poopmail client. This client will be discontinued as soon as the webinterface is available. This should be used for debugging only.

## Features

- Login
- List mailboxes
- Open mailboxes
- Create mailboxes
- Delete mailboxes
- View emails
